rstoolbox.components.FragmentFrame¶

class rstoolbox.components.FragmentFrame(*args, **kw)¶

The FragmentFrame extends pandas.DataFrame adding some functionalities in order to facilitate comparissons. Filled through the functions provided through this library, each row represents a position of a fragment, while columns describe the properties.

Methods

add_quality_measure(filename[, pdbfile]) Add to the data the RMSD quality measure provided by the r_fraq_qual app from Rosetta.
add_source_file(file) Adds a source file to the FragmentFrame.
has_source_file() Checks if there is a source file added.
is_comparable(df) Evaluate if the current FragmentFrame is comparable to the provided one.
make_sequence_matrix([frequency, round]) Generate a PSSM-like matrix from the fragments.
quick_consensus_sequence() Generate a consensus sequence as the most common representative for each position.
select_quantile([quantile]) Returns only the fragments under the rmsd threshold of the specified quantile.
