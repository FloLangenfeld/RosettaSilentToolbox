rstoolbox.analysis.cumulative¶

rstoolbox.analysis.cumulative(values, bins=100, max_count=None, upper_limit=None, cumulative=1)¶

Generates, for a given list of values, its cumulative distribution values.

Parameters:
  • values (list) – list of values to analyze.
  • bins (int) – number of bins in which to split the data (~resolution)
  • max_count (int) – max number of counts expected. If nothing is provided, defaults to the length of values.
  • upper_limit (numeric) – Max value for the bins. When not defined it is set to values.max(). When defined, is only taken into consideration when values.max() < upper_limit.
  • cumulative (int) – positive values mean do the cumulative (default); negative values mean do the inverted cumulative. 0 will provide non-cumulative distributions.
Returns:

Three lists: 1) the raw cumulative values, 2) precentage (0-1) cumulative values, and 3) bin positions. 2 and 3 would correspond to the y,x values of the plot.
